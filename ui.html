Vercel setup guide

  1. Push the api/rules.js file to your GitHub repo
  2. Import the repo in vercel.com
  3. Add environment variables in Vercel dashboard:
    GITHUB_TOKEN GITHUB_OWNER GITHUB_REPO GITHUB_FILE WRITE_SECRET
  4. Deploy, then paste the /api/rules URL above
